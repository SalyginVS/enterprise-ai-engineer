# Паспорт лаборатории на старте курса

**Дата фиксации:** 21.02.2026  
**Версия:** 1.0

---

## Аппаратная платформа

| Компонент | Значение |
|-----------|---------|
| CPU | Intel Core i9-9900KF (8C/16T), буст до 5 ГГц |
| RAM | 32 ГБ DDR4 |
| GPU | NVIDIA GeForce RTX 3090, 24 ГБ VRAM |
| SSD | ~1 ТБ, ext4, свободно ~880 ГБ |
| Сеть | Гигабитный Ethernet, MikroTik hAP ax2 |

## Программная платформа

| Компонент | Версия / Значение |
|-----------|-----------------|
| ОС | Ubuntu Server 24.04 LTS (headless) |
| NVIDIA Driver | серия 59x |
| Docker Engine | 28.x (поддержка --gpus all) |
| Ollama | 0.16.2 (systemd-сервис) |
| Ollama API | `127.0.0.1:11434` |

## Модельный стек

| Роль | Модель | Размер | Тип |
|------|--------|--------|-----|
| Быстрый универсал | `qwen3:8b` | ~5.2 ГБ | Text |
| Research / аналитика | `qwen3:14b` | ~9.3 ГБ | Text |
| Основной кодер | `deepseek-coder-v2:16b` | ~8.9 ГБ | Code |
| Reasoning-специалист | `deepseek-r1:14b` | ~9.0 ГБ | Text+Thinking |
| Тяжёлый кодер | `qwen3-coder:30b` | ~18 ГБ | Code |
| Мультимодальный | `qwen3-vl:8b` | ~6.1 ГБ | Vision+Text |

**Правило:** не запускать одновременно более одной модели 14B+ при активной нагрузке.

## Проверка работоспособности

Все модели проверены. Тест мультимодальной модели qwen3-vl:8b пройден успешно — корректно распознала изображение с онлайн-тестом (смартфон, карандаш, очки, секундомер).
